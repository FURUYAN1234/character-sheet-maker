## AI Character Sheet Maker v1.3.7

### What's New / 更新内容
- Updated Gemini image generation to the current Nano Banana 2 `gemini-3.1-flash-image` REST flow with `responseModalities: ["TEXT", "IMAGE"]`.
- Disabled rollback to legacy Gemini image preview model names.
- Updated OpenAI image generation for current `gpt-image-2`: `1024x1792`, `high`, `output_format: "png"`, and 600-second timeout.
- Added a 32,000-character OpenAI image prompt guard and MIME-aware image handling before watermarking.
- Rebuilt the OpenAI client module to remove syntax-breaking mojibake strings while keeping the same app-facing API.

## AIキャラクターシートメーカー v1.3.7

### 更新内容
